# Test Level 01

This is a test level for development and testing.

## Audio File Required

**Important**: You need to add an `audio.mp3` file to this directory.

Options:
1. Use any royalty-free 30-60 second instrumental MP3
2. Use a simple metronome or click track
3. Generate a simple test tone

The audio file should be named `audio.mp3` and placed in this directory.

For testing purposes, you can use any MP3 file - the beatmap has notes spaced every 2 seconds starting at 2 seconds in.
